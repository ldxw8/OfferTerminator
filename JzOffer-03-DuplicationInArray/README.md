## 题目描述
面试题 3：找出数组中重复的数字  
在一个长度为 n 的数组里的所有数字都在 0 到 n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

### 举例
```
Input: {2, 3, 1, 0, 2, 5, 3}
Output: 2 or 3
```
